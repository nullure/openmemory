Project Details
- Project: openmemoryost packages/core is a memory engine with ingest, recall, anchors, beliefs, sector routing, and waypoint-assisted retrieval.
- Store layer: MemoryStore implements anchors, beliefs, memory nodes, sector states, and waypoint edges with user scoping and cloning.
- Types: Anchors, beliefs, memory nodes, context packets, waypoint edges, and sector states define core data model.
- Embeddings: providers include Fake, OpenAI, and Siray; embeddings are sector-prefixed and normalized.
- Ingest pipeline: chunk/embedding, projection, sector routing, novelty gating, anchor creation, belief insertion, sector state updates.
- Recall pipeline: per-sector query embeddings, candidate scoring, optional waypoint expansion, ranked output, context packet.
- Waypoints: edges are created from recent nodes using semantic similarity, entity overlap, sector match, temporal adjacency with weighted score.
- Siray provider: OpenAI-compatible API with bearer token auth and base URL https://api.siray.ai/v1.
- Manual real-run script: packages/core/test/manual_siray_run.ts performs two ingests, builds waypoint edges, runs recall, and prints candidates/context.
- Sector taxonomy: core sector ids are episodic, semantic, procedural, emotional, reflective with semantic as default fallback.
- Tests: suite uses node --test with experimental type stripping.
